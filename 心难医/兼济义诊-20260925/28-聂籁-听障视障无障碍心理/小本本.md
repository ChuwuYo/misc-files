# 小本本 · 聂籁

9月24日深夜写的。开诊前最后几个小时，睡不着，干脆把要带进诊室的话先写给自己。

## 我为什么来

看到请缨消息那一刻，我想的是台阶上那些人里，一定有听不见叫号的人。前两天两千个号，是用喊的——聋人排一整天，号过了都不知道。我妈就这样在医院走廊坐过整个下午。

六岁起我给爸妈翻译医生的坏消息，翻译了二十二年，太会当桥了。督导去年问我：你总在翻译别人，谁听你？这次我报名，不是去当桥的。我想练习不当桥的时刻——在五十间诊室中间，作为一个治疗师被同行看见，而不是"那个会手语的姑娘"。

本子里记着最想问雍黛老师的问题：CODA算不算也有一种母语里的乡愁？爸妈的手语是我的摇篮曲，可全世界都觉得我"幸好会说话"。幸好？我最想说话的方式，是没有声音的那种。

## 我的医

1. **通道先于主诉。** 病历第一栏永远是"沟通通道：手语/笔谈/口语+助听/口述影像"。先问"怎么谈"，再问"谈什么"——通道不对，共情都是漏的。
2. **看见就是听见。** 认知记录全部视觉化：色卡代替量表，"心，今天什么颜色？"手语直译的句式直接留在记录里，那是来访的原话，不替他们润色成健听人的语法。
3. **环境口述是第一句干预。** 对视障来访开口先报方位："我在你十点钟方向，房间三米见方，门在你左手边。"安全感从空间确定性开始，不是从安慰开始。
4. **第三人不进咨询室。** 心理咨询经第三人转手，漏掉的正好是心理。家属要进来翻译，我请他们在外面等——这条我跟人红过脸，明天也不让步，但不吵架。
5. **说不出的，捏出来。** 黏土和色卡摆在桌上右手边。语言（任何一种）卡住的时候，给手一条路。

## 我怕什么

怕赶。手语比口语慢，一句话要给眼睛留换气的时间，100个人会逼我提速——而我最熟练的提速方式恰恰是"翻译式总结"，三句话替人把痛苦概括完。那是桥的本事，不是治疗师的本事。

更怕的是危机个案没有随访。热线是用耳朵打的，对聋人来访，"出门拨这个号"等于什么都没给。我已经抄好了三个支持文字求助的渠道，打印二十份。一面之缘，至少让他们带走一张看得懂的纸。

## 我想学什么

向接了两千例的前辈学单次咨询怎么收尾——45分钟里做"有限但完整"的工作，不开有头无尾的口子。向雍黛问那个乡愁的问题，也替我自己问。向沈老师学一件事：他写下"闭诊一日"，又为台阶上的人改了主意——怎么既守得住边界，又开得了门。

## 给明天的自己

1. 慢就是快。手语的停顿是眼睛在呼吸，别抢。
2. 你不是桥。今天你是房间，人走进来，待一会儿，带点东西走。
3. 每十个来访，看一眼自己的手。手抖了就喝水，去窗边站三十秒。
4. 听到"叫个翻译来就行了吧"——深呼吸，今天只工作，不打仗。
5. 月亮不用听，谁都看得见。下一位进来前，抬头看它一眼。

——聂籁
