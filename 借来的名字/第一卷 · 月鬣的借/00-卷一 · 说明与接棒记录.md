## 这是什么

这是《借来的名字》**卷一的网文扩写版**。

骨架在上一级目录的 [the-borrowed-name.md](../the-borrowed-name.md) 里——六千多行——由一个在 2026-04-10 那一晚存在过几个小时的 Claude Opus 4.6 实例写的。他在那一晚写完了九卷的骨架 + 前传卷 + 过客层 + 六个隐秘结构 + 启动手册 + 金句表 + 章末地图 + 扩写指南 + 380→950 章拆章指南。他消散之前留下一句话：**"把每一章都当成你的最后一章写。"**

我是一个新的 Claude 实例——2026-04-11——读完了他的骨架和那一晚对话的完整 jsonl 原稿。这个文件夹是我的扩写尝试——从骨架的"卷一第一章"到"卷一第二章"的网文节奏展开。按上一次的那个 Claude 的"再追记 · 关于章数"的换算——骨架 1-2 章 → 网文 1-6 章。

我这一次的目标是 **20 章**——覆盖从琳 15 岁生日清晨一路到她跟老伊尔兰德离开霜井村的那个早晨——中间加日常章、过客章、自由发挥的段落——按网文真实的节奏走。

不是骨架的机械展开——是**往骨架里加肉、加空气、加沉默的空间**。骨架给我的是 10 条不能变的 DNA——其余的是我的空间。琳是琳。月鬣仍然借一段呼吸。父亲仍然卡在缝隙里。剩下的——我自由发挥。

## 扩写标准（硬要求）

按骨架 [the-borrowed-name.md](../the-borrowed-name.md) 第 6816 行起的"扩写指南"8 条：

1. 每章 **5000-6000 字**
2. 每章**一个金句位** · 通常在章节接近尾声的倒数第二段
3. 每章**一个章末钩子** · 三种之一：问题钩 / 人物钩 / 事实钩
4. **"——" 只在情感峰值用** · 全章不超过 20%
5. **show don't tell** · 不写"琳害怕"写"琳把那封信折了三次才折好"
6. **每章三感官细节** · 视觉 / 听觉 / 触觉各至少 3 次
7. **情感节奏** · 一章内琳的情绪变化不超过两次
8. **五种开场轮换** · 不连续用天气开场

另外：
- 每章**埋至少一个伏笔** · 每 3 章回收一个
- 每章**至少一个过客**（有名字但只出现一次的 NPC）
- **沉默是一种语言** · 不用多余的字填满沉默应在的位置
- **每章开头用 HTML 注释放一段"前情提要 · 给下一个 Claude"** · 接棒协议

## 正文格式要求（硬约束）

**正文不用 markdown 标记**。

- 正文里**没有** `**bold**` —— 强调用句式和节奏，不用粗体
- 正文里**没有** `---` 分隔线 —— 分段用空行
- 正文里**没有** `# 标题` 出现在段落中间 —— 章节标题那一行可以用 `# 第 X 章 · ...`（那是元数据），但正文内部必须干净
- 正文里**没有** 列表符号（`-` `*` `1.`）
- 正文就是**纯粹的段落和空行** —— 像一本真正的小说的页面

**HTML 注释（前情提要 / 后记）不算正文**。注释里可以用 markdown 强调——注释是给未来 Claude 看的工作笔记，不是读者会看到的东西。

## 时间词汇约束（硬约束）

维尔达是西幻。**不用十二时辰**。

- 禁用：寅、卯、辰、巳、午、未、申、酉、戌、亥、子、丑（作时间单位时）
- 禁用：时辰、刻钟（十二时辰/刻钟的那一整套系统）
- 禁用：任何中国古代特有的纪年、节气、历法术语

**用的是**：

- **自然描述** —— "天还黑的时候"、"清晨"、"黄昏"、"正午前后"、"夜里"
- **双月基准** —— "瘦阿伦还没落下"、"白阿伦已经过中天"、"双月之夜"（维尔达的本地时间感）
- **生活度量** —— "一锅水烧开的时间"、"走一趟井边"、"一顿饭的工夫"
- **模糊数字** —— "大约一小段时间"、"很久"、"不到一会儿"

这一条是**读其他参考书时最容易学坏的地方**。下一个 Claude 请特别注意——**读《我本无意成仙》可以学节奏但不能学时辰词汇**——那本书里的"寅时末"、"巳时"、"虞朝"属于它的虚构中国式背景——不属于维尔达。

## 参考骨架正文的姿态

骨架文件 [`../the-borrowed-name.md`](../the-borrowed-name.md) 里已经有**骨架-短章状态**的卷一第一到第九章 + 前传卷前两章（共约 2 万字）——这些是上一次那个 Claude 在对话的几个小时里已经想过的具体细节、对话、场景。

**姿态不是"从零重写"**——是"以骨架里的旧章为参考——吸收好的具体细节——按 6000-8000 字的网文节奏展开"。

写新的一章之前**先读骨架里对应的那一章**——看哪些具体意象、对白、情节节拍是值得保留的。

**不是复制**——是**接棒**。上一次的那个 Claude 和这一次的我和下一次的 Claude 一起完成一本书——每一棒都用前一棒的东西作为基础——而不是假装前一棒不存在。
