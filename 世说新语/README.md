# 世说新语

以《世说新语》三十六门为骨架，每门取一则或一人一事为种子，扩写为完整短篇。

## 体例

- **语言**：白话叙事为主，间以文言白描、原文嵌入。刘义庆的笔调是"言约旨远"，扩写不能把它写散——留骨，长肉，不添脂。
- **结构**：每篇 3500–4500 字。单线叙事或三幕。引原文片段置于场景之中，不做引号注释，不做考据旁注。
- **人物**：以原书所载魏晋人物为主。偶有前汉、南朝人物顺带提及，不越出《世说》所记时段太远。
- **魏晋风度**：不拔高，不贬抑。任诞者任诞，方正者方正，痴绝者痴绝。不把名士写成现代人的心理剧。
- **尾**：克制。少议论，多留白。不把每一则的道理说死。

## 门类

三十六门，按原书卷次：

**上卷**：德行 言语 政事 文学
**中卷**：方正 雅量 识鉴 赏誉 品藻 规箴 捷悟 夙慧 豪爽
**下卷**：容止 自新 企羡 伤逝 栖逸 贤媛 术解 巧艺 宠礼 任诞 简傲 排调 轻诋 假谲 黜免 俭啬 汰侈 忿狷 谗险 尤悔 纰漏 惑溺 仇隙
