## 序言

### 致绝对小白读者：怎么用这本书

先把丑话说在前面：**这本书原本不是写给完全零基础的人**。它的默认读者是"已经会写一点代码、想往 AI 应用方向跨"的工程师。打开觉得密度大、术语多、节奏快是正常的，不是你笨。

只要你愿意补两块前置，照样能啃下来：

1. **先离开这本书 1-2 周**，把 Python 基础语法过一遍。两个免费资源任选一个，看到能自己写一个百行小脚本（读 CSV、调 HTTP API、存 JSON）就够：
   - 廖雪峰《Python 教程》：<https://liaoxuefeng.com/books/python/introduction/index.html>（中文，看到"函数式编程"为止）
   - Eric Matthes《Python Crash Course》（中文版《Python 编程：从入门到实践》，第 1-11 章）
2. **再花 30 分钟**学几条 Linux/macOS 命令行：`cd / ls / mkdir / cat`、`python -m venv`、`pip install`、`git clone`。会这些就能把书里的代码跑起来。
3. **回来按下面"路径 A"读**，但允许自己**先跳过两章**：
   - 第 03 章（数学）——只看每节开头的"先记住这三件事"速记盒，公式推导整段跳过，不影响后面学 RAG / Agent。
   - 第 09 章（手搓 Transformer）——只读第一遍的概念路径（章首给了明确指引），200 行代码留到将来想往深里走时再回来。

读法分三档，自己对号入座：

- **绝对小白**（没写过项目、没用过命令行）：先做上面 1+2，然后路径 A，跳 03 / 09，预计 5-7 个月。
- **半小白**（会一点 Python，没做过工程项目）：直接路径 A，03 / 09 选读，预计 4-6 个月。
- **转型工程师 / 应届生**（有后端或数据经验）：路径 B，6-10 周。

这一行不缺看教程的人，缺的是真把代码跑通、把坑踩过一遍的人。你慢一点没关系，别停。

### 实操预算与环境

小白书不会告诉你的三个卡点：API key 怎么搞、网络怎么通、GPU 钱从哪出。这一节一次说清楚。

**1. API Key 与账号**

跑完书里所有 LLM 调用代码，至少要一家 API。三个层级任选其一：

- **DeepSeek**（强烈推荐入门）：国内最便宜，¥1/100 万 token 起步，支付宝 / 微信充值即用，官网注册到拿到 key 不超过 30 秒。第一次跑代码 ¥50 充值就够把前 17 章所有 demo 跑完一遍，跑完还能剩。
- **OpenAI**：生态最全、文档最全、SDK 最稳，但需要海外信用卡 + 海外手机号。最贵的一档。
- **Anthropic（Claude）**：同样需要海外信用卡，价格与 OpenAI 同档。

国产兜底（都支持微信 / 支付宝、都有 OpenAI 兼容接口）：通义千问、智谱 GLM、Moonshot Kimi、火山方舟（豆包）。任意一家都能替代 OpenAI / Claude 跑完本书 80% 以上的代码，仅在第 14 章 Prompt 跨模型对比、第 24 章 LLM-as-judge 时会感觉到差距。

**2. 网络访问**

境内访问 OpenAI / Anthropic / Hugging Face / GitHub 需要稳定的网络方案，**本书默认你已经能解决，不展开方法**（合规问题各人各自处理）。如果你暂时解决不了，下面这套替代方案足以读完全书：

- Hugging Face 模型下载 → 用 `hf-mirror.com` 镜像，或全量切到阿里 ModelScope。
- OpenAI / Claude 调用 → 全部换成 DeepSeek / 通义 / 智谱，本书代码 99% 兼容 OpenAI SDK 调用风格，只改 `base_url` 和 `api_key` 两行。
- GitHub clone 慢 → 用 `https://ghproxy.com/` 加速或 Gitee 镜像。

**3. GPU 预算（按章节列）**

| 章节 | 算力需求 | 花费 |
| --- | --- | --- |
| 01-07（基础 + 经典 ML） | CPU 即可 | 0 元 |
| 08-10（PyTorch + 训练技巧） | CPU 能跑 MNIST，想跑得快用 Colab 免费 T4 ~12GB | 0 元 |
| 11-12（LLM 原理 + 开源生态） | CPU + 7B 量化 + Ollama 本地跑（M2 Mac 8GB+ 或带 8GB+ GPU 的 PC） | 0 元 |
| 13（微调） | 必须 GPU。Colab Pro $10/月（单次微调 ~$1-2）/ RunPod 4090 ¥4-6/小时 / Modal 按秒计费 | LoRA 端到端一次 ¥10-30 |
| 14-19（应用层） | CPU 即可，全靠 API | 包含在 API 预算里 |
| 20-22（生产工程） | 本地 Docker + 一台 4090 复现 90%；剩下 10% 同 13 章租卡 | 同上 |
| 23-27 | CPU 即可 | 0 元 |

**4. 跑通全书的总预算**

一次性投入 **¥50-200**（API ¥30-50 + 1-2 次 GPU 微调 ¥20-150）。不打算自己跑微调、只用 API 看效果的，**¥30 以内**就能把全书代码过一遍。

**5. 省钱铁律（也是工程铁律）**

- 永远先在 DeepSeek 上做调试和实验，OpenAI / Claude 只在生产前的模型对比环节调用一次。一次 prompt 迭代往往要试 50 遍，省下的就是几十倍价差。
- 第 22 章会教你开 prompt caching 和 batch API，能再砍 50-90% 的账单。前期养成习惯，后期不掉头发。
- 用 Ollama 在本地跑 7B 量化模型，能 0 元解决 50% 的「我就想试一下这个 prompt 行不行」需求。试通了再上 API。

读完这一节你会发现，跑通本书全部代码的现金门槛比一顿火锅贵不了多少。真正贵的是时间，那笔账自己心里有数。

### 这本书是为谁写的

- **正在转型的后端 / 前端 / 数据工程师**：你已经会写代码、会上线服务，缺的是 LLM、RAG、Agent、微调、向量库、部署的那一整套技能拼图。
- **刚毕业、想进 AI 行业的工程师**：你不缺学习能力，缺的是一份知道哪里是重点、哪里是噪音的完整路径。
- **正在做 AI 应用、但卡在某个环节的工程师**：评估上不去、成本压不下、Agent 进死循环、向量库选错——本书每一章都是为某个真实的工程问题写的。
- **想从算法工程师 / 数据科学家迁移到「AI 应用工程师」的人**：你懂模型，缺的是工程化、产品化、运维化、合规化的全套能力。

不适合：
- 想转 LLM 预训练 / 后训练研究的人——那是另一条路。
- 想做底层框架（PyTorch、vLLM、CUDA kernel）的人——这本书会用，不会教你写。
- 只想跟潮流写一两个 demo 给老板看的人——本书每章 1-2 万字，读完是真的辛苦。

读这本书前你应该已经会的（书里不再回头补）：
- Python 基本语法 + 至少写过一个独立可运行脚本。**本书不教 Python 基础语法**，零基础同学请先看上一节"致绝对小白读者"给的资源，或官方教程 <https://docs.python.org/zh-cn/3/tutorial/index.html>。
- Linux / macOS 命令行基本操作、能装 Python、能跑 `git clone`。
- 至少了解一种 Web 服务（FastAPI / Flask / Express 任一）的请求-响应模型。
- 知道什么是 JSON、HTTP 状态码、环境变量，会读官方文档而不是只看二手教程。

**特别提醒**：第 02 章是「Python 工程化」，不是「Python 入门」——它假设你 `def`、`class`、`import`、`with`、`try/except` 都写过，只是没用过 ruff、mypy、pydantic 这套工程工具链。基础语法没过关，先补再来，省得在第 02 章前几页就劝退自己。
