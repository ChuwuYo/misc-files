# v0.12 — 行业 / 市值中性化（Asness 灵魂拷问）

**日期**: 2026-04-21
**作者**: claude (loop iteration #12)

## 动机

v0.11 walk-forward 上 ortho+ewma 因子 mean IC=+0.044, agg_t=+3.64 看似漂亮。
但 Asness/Frazzini (2013) "The Devil in HML's Details" 警告：**很多看似 alpha 的
信号其实是 70% size/sector 暴露的副产品**。
v0.12 用 Barra 风格中性化做"灵魂拷问"——剥离 sector 和 size 后还剩多少 alpha。

研究：[research/v0.12_neutralization.md](../research/v0.12_neutralization.md)

### 2. 新建 `src/neutralize.py`

```python
sector_neutralize(factor, sector_map)
    # 截面 within-sector demean (== OLS on dummies, no intercept, residual)

size_neutralize(factor, size_map)
    # 截面 OLS: factor ~ log_market_cap, return residual
    # 闭式解 β = sum(x*ε) / sum(x²)，向量化

```

### 3. `compute_pool_factor` 增 `sector_map` / `size_map` 参数

放在 EWMA 之后、最终 cs_rank 之前。中性化后再做一次截面 rank 把输出压回 [-1, 1]。

## 关键洞见

### 1. 67% IC 来自 sector exposure
sector 中性化把 mean IC 从 +0.037 砍到 +0.012——**信号 2/3 是行业切换 timing
（Tech vs Energy 的相对强弱），1/3 是 within-sector alpha**。

这与 Asness 1999 综述吻合："对量化因子做行业中性后，估计 30-50% 的 alpha 消失"。
我们 67% 损失偏高，原因：SP50 universe 小（51 只），sector 仅 8 类，
within-sector demeaning 留下的自由度少（如 Energy 只有 2 只，demean 几乎是反号）。

### 2. Size effect 较小
size 中性化只损失 22%（+0.037 → +0.029）。
合理：SP50 都是大盘股，跨股 size 跨度虽大（85x），但都在"大盘"范畴内，
传统的 small-cap effect 在此 universe 不强。

### 3. 双中性化几乎清零
sector + size 中性化后 mean IC=+0.007, t=+0.55——**纯 alpha 已不显著**。
诚实结论：**我们的因子主要是 sector + size 风格暴露的代理**，不是 stock-picking alpha。

### 4. h=20 受影响最大
sector neutralize 在 h=20 上把 IC 从 +0.041 砍到 +0.011（损失 73%）——
长 horizon 信号更"风格化"，短 horizon 还有部分微观结构 alpha。

## 重新理解因子

| 视角 | 结论 |
|---|---|
| 整体表现（含暴露） | mean IC +0.044, agg_t +3.64 — 优秀 |
| **纯 alpha（中性化后）** | mean IC +0.007, t<1 — **不显著** |
| 用途 | sector rotation timing > stock picking |

## 副作用 / 已知限制

1. **sector demean 损失自由度**：8 sector × 51 sym 平均 6.4 sym/sector，
   小 sector（如 Energy 2 只）demean 几乎是 zero-sum——人为引入噪声
2. **没有 beta 中性化**：market beta 暴露未剔除（需 SPY 数据）
3. **没有 momentum factor 中性化**：经典 4 因子 / 5 因子只覆盖一半
4. **size 用静态 marketCap**：动态 size（quarter-end）更准确
5. **GICS sector 不是最细粒度**：industry 级（GICS 4-level）更精，本版本未做

## 实务含义

### 选项 A：接受 + 包装
我们的因子是"sector rotation + small alpha"——历史上 sector rotation 是
合法的多策略组合一部分。可作为 ensemble 的一员，与纯 stock-picking 因子互补。

### 选项 B：升级为 within-sector 因子
重写 `compute_pool_factor` 让 ic_weight 在每个 sector 内独立学权重 + 排名，
然后跨 sector 加总。期望：纯 alpha IC 提升到 +0.02 以上。
**留 v0.13 探索**。

### 选项 C：扩 universe
SP50 太小，sector 自由度受限。扩到 SP500 / Russell 1000 后 within-sector
样本足，中性化损失会小。但 yfinance 大批量数据稳定性差。
