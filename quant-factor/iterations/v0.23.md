import pandas as pd

alphas = ALPHA_GROUPS['all']

idx = pnl_ic.index.union(pnl_lgbm.index)

## 为什么之前 LGBM 配置失败

回头看，在 v0.21/v0.22 我一直用 `rank_features=True`。rank 特征相当于：
- 所有 alpha 输出先 percentile rank，强制归一到 [-0.5, 0.5]
- 极值信号被压到 ±0.5 附近
- LGBM 可切分点变少（信息丢失）

把 raw features 传给 LGBM：
- klen=0.05 和 klen=0.001 的差异保留
- LGBM 能学到 "klen 很大 AND funding 很大 → 强 signal"
- target 仍 rank 化防止 return 极值干扰

**这是 v0.22 没走到的那一步**。

## 副作用 / 已知限制

1. 92/8 min fold +0.05 非常贴近阈值——小数据变化可能突破 0
