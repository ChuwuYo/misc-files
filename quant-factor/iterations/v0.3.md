## 改动

### 1. `src/timeframes.py` — 时间框架抽象

```python
TimeFrame.{M1, M5, M15, M30, H1, H4, D1, W1, MO1}
TimeFrame.short() / .long() / .all()
ANNUALIZATION[tf]
resample_panel(panel, tf)         # OHLCV-aware aggregation
cross_tf_corr(signals_by_tf)      # 跨 TF 信号相关性矩阵（冗余检测）
horizon_for_tf(tf, days)          # 跨 TF 的可比 horizon
```

**关键修复**：pandas resample 在空 bin（周末）对 `volume: sum` 返 0 而非 NaN，
导致 `dropna(how='all')` 失效。改用 `dropna(subset=['close'])` 后 D1 重采样产出
正确的 252 个交易日（之前错产 365 个日历日导致下游全 NaN）。

### 2. `src/data.py` — 多尺度合成数据生成器

新增 `sample_intraday_panel(n_days, n_symbols, bar_minutes, bars_per_day, scale_strengths)`。
`scale_strengths` 在多个显式 horizon（默认 3/12/48/240 个 5min 单位 = M15/H1/H4/D1）
注入截面反转。聚合到不同 TF 后信号方向可能翻转——这是真实数据中也常见的现象。

### 3. `src/factor.py` — `compute_mtf_factor`

```python
compute_mtf_factor(
    panel, timeframes=[M5, M15, H1, D1],
    window_per_tf=5, vol_alpha=1.0,
    combiner='ic_weight',      # 'equal' or 'ic_weight'
    ic_lookback=60,
)
```

每个 TF 独立跑 v0.2 流水线 → 对齐到最细 TF（HTF 信号 ffill）→ 组合：

- **equal**：z-score 后等权平均
- **ic_weight**：滚动 IC 加权，**带符号**——负 IC 的 TF 自动被反向计入（关键设计）

**Lookahead-bias 修复**：rolling IC 在 t 时刻包含 t+1 的 forward return，
所以 weights 必须 `shift(1)`，确保 t 时刻的权重只用 t-1 及以前的信息。

## 解读

### 关键洞见 1：信号符号不一致是 MTF 的核心难点
合成数据在 M15/H1 注入了反转，但聚合到 H4 后变成动量（跨日相关性反转）。
**equal 权重必失败**（强行平均矛盾信号）。**ic_weight signed 自动学到符号**，
把"反着走"的 TF 反着用。

### 关键洞见 2：MTF 的真正价值是降噪 + 减换手
- IC 提升 5x 是其次，更重要的是 **Turnover 22% → 5%**（成本降低 4 倍）。
- 多 TF 融合天然平滑了高频噪声——HTF 信号慢变，把手频率天然降低。

### 关键洞见 3：IS = OOS 说明无过拟合
ic_weight 用 60 bar lookback 学权重；OOS 60 天与 IS 60 天 IC 几乎相同（+0.154 vs +0.158）。
Lookback 学到的是**结构性符号关系**而非噪声——稳健。

## 性能

- 5 TF MTF 计算 460,800 行约 600ms。瓶颈在 resample 的 per-symbol 循环。
- v0.10 拟用 polars / numba 加速 resample 阶段，目标 < 200ms。

## 副作用 / 已知限制

1. **资金线/滑点未模拟**：T-B 年化是理想化数字。
2. **synthetic data 注入信号过于干净**：真实市场反转 IC 通常 0.02-0.05，
   合成的 0.15+ 不可外推。但**相对提升**（MTF vs single-TF 5x）可外推。
3. **最细 TF 决定输出频率**：MTF 输出在 finest TF 网格上。如果 finest=M5 但要日频
   下单，需在调用方做 last-bar-of-day 选取。
4. **ic_lookback=60 写死**：小样本起步阶段 fallback 到等权——但等权 IC 已显示是
   错的方向。冷启动 30 bar 内不要交易。
5. W1 / MO1 长线 TF 在合成数据上数据点过少没法评估——v0.7 真数据后再测。
