# v0.2 — 特质波动率阻尼 + 模块化流水线

**日期**: 2026-04-21
**作者**: claude (loop iteration #2)

## 动机

v0.1 单一函数难扩展。同时低波动异象（AHXZ 2006）+ BAB 设计模式（FP 2014）
提示我们：高波动股的反转信号噪声大，应被**降权**而非简单纳入。

研究笔记见 [research/v0.2_idio_vol.md](../research/v0.2_idio_vol.md)。

## 改动

### 1. 模块化 — 新增 `src/pipeline.py`

定义 `Pipeline` 容器 + 可拼接 `Stage` 函数。借鉴 Zipline Pipeline / Qlib Handler 设计。
当前 stage 库：`reversal_signal` / `winsorize` / `vol_dampen` / `cs_rank` / `cs_zscore` / `neg`。
共享 `ctx` 字典缓存中间量（如 log return），避免重复计算。

### 2. 新增 `vol_dampen(vol_window, alpha)` stage

```
weight = 1 / (1 + α · vol_rank_pct)
signal_out = signal_in × weight
```

- α=0 关闭阻尼，恢复 v0.1
- α=1 默认：高波动股权重 ~0.5，低波动股权重 ~1
- α=2 强阻尼：高波动股权重 ~0.33

### 3. 加入 `winsorize(0.01, 0.99)` stage 控制截面极端值

### 4. `compute_factor()` 接口向后兼容（额外 kwargs 默认值给出 v0.2 行为；`vol_alpha=0` 完全退回 v0.1）

## 测得指标（合成数据 252d × 500 symbols, seed=42, window=5, horizon=1）

| 配置 | IC | IR | T-B 年化 | Turnover |
|---|---:|---:|---:|---:|
| v0.1 baseline (α=0) | +0.0048 | **+1.437** | +5.72% | 20.96% |
| v0.2 mild (α=0.5) | +0.0050 | +1.535 | +3.67% | 20.93% |
| v0.2 default (α=1.0) | +0.0051 | +1.596 | +6.79% | 20.92% |
| v0.2 strong (α=2.0) | +0.0052 | **+1.686** | +7.05% | 20.92% |
| v0.2 vw=10 (α=1.0) | **+0.0055** | +1.725 | +5.84% | 20.95% |
| v0.2 vw=60 (α=1.0) | +0.0027 | +0.847 | -1.09% | 20.89% |

**核心结论**：
- IR 随 α 单调上升（+1.44 → +1.69，**+17%**）。验证假设：高波动股反转噪声更大。
- 短 vol 窗口（10 日）反而最佳，因为 vol 是动态变化的。
- 长 vol 窗口（60 日）严重退化——信号严重滞后于真实波动状态。

性能：126k 行 32ms（含 vol 计算），仍远低于 2s 预算。

## 副作用 / 已知限制

1. v0.2 pipeline 加入 winsorize 后，α=0 baseline 比 v0.1 单纯版本的 IR 略低（1.44 vs 1.62）。
   原因：合成数据本就无极端值，winsorize 误伤了少量真信号。**真数据上 winsorize 应是净正**——保留。
2. vol_dampen 用 raw vol，不是真特质 vol（IVOL）。等 v0.3 加因子残差后升级。
3. vol_window 选择敏感：合成数据上 vw=10 最佳，但真数据上很可能不同——v0.x 加超参数自适应。

## 与开源因子的对照

| 维度 | 本 v0.2 | Qlib Alpha158 | WorldQuant Alpha101 |
|---|---|---|---|
| 反转信号 | log_ret 累加 + 截面 rank | ROC5/10 + STD20 比值 | rank(close-mean) |
| 波动率使用 | 作为 dampening 权重 | 作为独立特征 | 通常作为 scale 算子分母 |
| 模块化 | Pipeline 容器 | DataLoader + Handler | 单行表达式 |

我们走的是「显式模块 + 经济直觉权重」路线，更易解释、更易审计；
代价是不能像 Alpha101 那样暴力枚举公式。
