# v0.12 研究笔记 — 行业 / 市值中性化（Barra 风格）

## 动机

walk-forward 上 ortho+ewma 因子 mean IC=+0.044 看似不错，但**未排除**：
- 信号是不是单纯偏向某个行业（如科技股集中）的 sector beta
- 信号是不是单纯偏向小盘 / 大盘的 size effect

如果中性化后 IC 暴跌 → 信号是风格暴露的副产品；
如果中性化后 IC 仍存 → 是真 alpha。

## 核心文献

### Barra (MSCI) USE3 / CNE5 风险模型
- 把股票收益分解为：market β + style factors（size / value / momentum / vol / ...）
  + industry dummies + idio
- 因子开发的标准做法：先**回归剔除**已知风险源，再看残差是不是预测下期收益

### Fama-MacBeth (1973) 截面回归
- 每日做截面回归：r_t = α + β1*size + β2*sector_dummies + ε
- 用残差 ε 作为"中性化后"的信号
- 时间序列上对 α 做 t 检验

### Asness, Frazzini (2013) "The Devil in HML's Details"
- 强调"风格暴露 vs alpha"分离的重要性
- 一个看似强的信号经常 70% 是 size/momentum exposure

## 期望结果

按 Barra 经验：
- 反转 / 短期信号的中性化损失通常 30-50%（但残差更稳定，IR 可能升）
- 如果中性化后 IC 减半但 IR 不变 → "差不多 = 风格暴露 + alpha 各半"
- 如果 IC 全失 → 信号是纯 sector momentum
