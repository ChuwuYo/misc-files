# v0.9 研究笔记 — PCA 去噪 + EWMA 平滑

## v0.8 遗留问题

- ortho 后 OOS h=10 IC=+0.019 t=+1.87 已接近显著
- **但 turnover 30%**——每天换 1/3 仓位，真实成本下吃光 alpha
- 原因：(1) 残差信号本身波动大；(2) ic_weight 权重每天重学

## 解法 1：PCA 去噪（Random Matrix Theory）

### Marchenko-Pastur 上界
对 N×T 信号矩阵（N 信号，T 时间），随机矩阵理论给出特征值上界：
```
λ_max_random = σ² * (1 + √(N/T))²
```
任何小于此上界的特征值都是噪声——只保留大特征值对应的主成分。

### Ledoit-Wolf (2004) 协方差收缩
- 样本协方差矩阵在 N≈T 时极不稳定
- 收缩到对角矩阵：`Σ_shrunk = (1-α)*Σ_sample + α*diag(σ²)`
- 用最优 α（解析解）

### 应用到 v0.8 ortho 残差
- 把 N=34 个残差信号作为列，T=753 天作为行
- 求协方差，做 PCA，保留前 K 个主成分（K = MP-bound 启发的截断）
- 用主成分作为新池子，再过 ic_weight

或更激进：用前 K 主成分**直接**作为 K 个新 alpha，绕过 combiner。

## 解法 2：EWMA 信号平滑

### 思想
最终 factor 是每日截面 rank。直接取每日 rank → 高频抖动。
对 factor 做指数加权平滑：
```
factor_smooth[t] = (1-λ)*factor[t] + λ*factor_smooth[t-1]
```
- λ=0：不平滑（v0.8 现状）
- λ=0.7：今天 30% + 昨天 70%——低 turnover
- λ→1：完全用历史，丧失新信息

### 与 ic_weight lookback 的区别
ic_weight lookback 平滑的是**权重**；EWMA 平滑的是**最终因子值**。
二者正交，可叠加。

### 经济直觉
真实因子信号衰减半衰期 5-10 天（v0.8 半衰期 ≈ 10d）——
今天的 rank 与昨天的 rank 有 ~70% overlap 的真信号，30% 是噪声。
EWMA λ ≈ 0.7 大致对应这个比例。
