# v0.2 研究笔记 — 低波动异象 / 特质波动率

## 核心文献

### Ang, Hodrick, Xing, Zhang (2006) — "The Cross-Section of Volatility and Expected Returns" (JF)

发现：高特质波动率（idiosyncratic volatility, IVOL）股票未来收益**异常地低**。
即"低波动异象"（low-vol anomaly）。学术上挑战 CAPM。

测度方式：用 Fama-French 三因子回归取残差，对残差做滚动 21 日（月度）标准差 → IVOL。

工程简化：在没有因子模型的早期阶段，**用 raw return 的 rolling std 近似** total vol，相关性 ~0.85+。

### Frazzini & Pedersen (2014) — "Betting Against Beta" (JFE)

发现：高 beta 股回报率被高估。AQR 旗舰策略 BAB 即按 beta 排序的多空组合。
设计原则：杠杆约束下，投资者追逐高 beta，导致定价偏差。

启发：把波动率作为**信号增强权重**比作为**独立信号**收益更稳定（因为反转/动量信号在低波动股上更干净）。

### Qlib Alpha158 中的波动率系列

```
STD5, STD10, STD20, STD30, STD60       # 滚动收益率标准差
HIGH0/CLOSE                             # 高低距离 / 收盘价
RSV5, RSV10, ...                        # Raw Stochastic Value
```

工程经验：STD20 是最常用的 vol proxy；STD5 噪声大但反应快；二者**比值** STD5/STD20 是波动率冲击信号。

## 设计选择

v0.2 引入两层增强：

1. **特质波动率 proxy**：用 20 日收益率 rolling std 作为简化 IVOL（v0.3 接入因子残差后再升级）
2. **信号 × inv(vol) 加权**：反转分数乘以 1/(1+vol_rank)，让低波动股的反转信号被放大、高波动股被抑制
3. **模块化**：把 v0.1 的单一函数拆成 `pipeline.py` 的可拼接 stage（`Signal` / `Filter` / `Combiner`），为 v0.4 多窗口融合做准备

## 预期效果

合成数据上 IC 提升应有限（合成数据各股 vol 是均匀采样的，没有低波异象注入）。
**但 IC 标准差应下降**（IR 上升），因为高波动股的反转噪声被抑制。
真实数据上预期收益更明显。

## 不做

- 不实现完整 Fama-French 残差回归（v0.3 与中性化合并做）
- 不引入 beta（需要市场指数，留待真实数据接入）
