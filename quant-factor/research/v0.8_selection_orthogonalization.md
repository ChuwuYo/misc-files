# v0.8 研究笔记 — 信号选择 + 正交化

## v0.7 问题诊断

真数据上 26 slots 池子 IC=-0.012，单 alpha klen+alpha001 IC=+0.015。
**combiner 把好信号稀释成噪声**。原因：
1. 大多数 alpha 在真数据上 |t|<1.5，individually no-signal
2. ic_weight 用 60-bar rolling IC 估计权重——估计本身有噪声 σ=O(1/√60)≈0.13
3. L1 归一化强制把权重分散到所有 slot
4. KBAR 之间高度相关（kmid 与 kmid2 相关 ~0.9），重复计入

## 解法 1：信号选择（Signal Selection）

### Sullivan/Timmermann/White (1999) "Data-Snooping, Technical Trading"
- 用 reality check 控制多重检验偏差
- 对我们：在 selection 阶段用 warmup 期 t-stat 阈值

### Hou-Xue-Zhang (2015) q-factor model
- 4 因子模型用**完整样本**测真实 IR；推到我们：
- 用前 50% 样本作为 warmup 计算每个 alpha 的 |t|，仅保留 |t| > 阈值的进入 pool

### Bonferroni 校正
- N 个 alpha 同时检验，p 值需乘以 N 才显著
- 我们 17 个 alpha → 0.05 / 17 ≈ 0.003，对应 |t| > 2.93
- 太严苛——折中用 |t| > 2.0 + 选定上限（top-K=5）

## 解法 2：正交化（Orthogonalization）

### Gram-Schmidt 序贯正交
- 按 t-stat 降序排列
- 每个新 alpha 减去其在已选 alphas 上的截面回归残差
- 优点：保留单 alpha 可解释性
- 缺点：序列依赖，第一个 alpha 决定结果

### PCA 主成分
- 对所有 alpha 做截面 z-score 后做 PCA
- 取前 K 个主成分作为新因子
- 优点：完全去相关
- 缺点：失去可解释性

## 解法 3：稳健的 ic_weight 改进

- 用 **sign(IC)** 而不是 IC 值——避免 |IC| 漂移导致权重剧烈变化
- 加 **t-stat shrinkage**：weight ∝ |IC| × shrinkage(|t|)
- 用 **expanding window** 而不是 rolling 60-bar——更长样本更稳
