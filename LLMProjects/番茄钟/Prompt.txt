生成一个功能完备、视觉美观、交互流畅的 HTML 番茄钟网页，整体风格为像素小清新 + 二次元像素艺术，你可以自由发挥设计与实现细节，但需满足以下基本要求：

必备功能：
- 番茄钟计时器：25分钟工作 + 5分钟短休息 + 15分钟长休息
- 基础操作：开始、暂停、重置、跳过当前时段
- 自动循环：工作 → 休息 → 下一个工作
- 完成计数：显示当天完成的番茄钟数量

设置功能（建议实现，鼓励扩展）：
- 各阶段时长可调节（工作、短休、长休）
- 音效提示（开始、结束、提醒），音量可调
- 自动播放下一个时段开关
- 你可以自由添加更多设置项，只要合理有趣

视觉设计（鼓励创意）：
- 像素风格 UI：8-bit 字体、像素按钮、像素进度条
- 清新配色：柔和的糖果色系为主，可以有浅色暗色主题切换和色系主题切换
- 响应式布局：适配桌面和移动端
- 页面标题直接显示“番茄钟”，不要副标题
- 你可以加入像素角色、动画、背景音乐、二次元元素等增强趣味性

技术要求：
- 单一 HTML 文件，包含所有 CSS 和 JavaScript
- 使用本地存储保存用户设置和统计数据
- 键盘快捷键：空格键开始/暂停，R键重置，S键跳过
- 页面标题动态显示剩余时间（运行中时）

开发流程：
1. 先生成一个详细的规格说明文档（spec.md），自由规划功能模块、UI设计、技术实现
2. 基于规格说明实现完整的番茄钟应用
3. 确保所有功能正常运行，界面美观易用
