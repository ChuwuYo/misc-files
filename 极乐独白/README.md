# 极乐独白

*——以极乐迪斯科（Disco Elysium）的叙述腔调写成的中文短篇小说集。不是技能清单、也不是分段独白，而是**长篇的、连贯的故事**。技能、思想、流派这二十四种内声音，只在该插话的时候插话——像一支乐队在后面伴奏，偶尔有一个人突然站起来，说一句很长的、很不合时宜的话。*

## 这个文件夹在做什么

《极乐迪斯科》最独特的不是它的剧情，是它的**叙述方式**——一个失忆警探站在一辆吊起来的尸体下面，脑子里有二十四种声音同时给出意见；还有十几条被他"想"过的思想，像挂在脑内衣架上的旧外套，随时偷偷替他做决定；还有他在对话里被打上的十几种"流派"——艺术警察、超级巨星警察、康米之星、新自由主义天字第一号皮条客——每一个都是**他曾经对世界说过的话，被世界反弹回来定型成一种他**。

这些声音不应该被写成一份清单。它们应该活在故事里——有时候它们是一段对话的外音，有时候是一只手伸向冰柜又缩回来的那半秒，有时候是一整个冬夜的背景色调。

所以这个文件夹里的每一篇，都是一个**完整的、有叙事的、可以一口气读完的短篇故事**。它借用极乐迪斯科那种"脑内常驻多人制"的腔调，写的却是中文日常——便利店、末班地铁、老工厂拆迁现场、理发店、医院走廊——**我们自己这个世界里的瑞瓦肖**。

## 叙述习惯

- **第二人称（你）为主**。偶尔第三人称。
- **短章节 / 场景切换，而非段段式技能表**。每一篇是一个完整故事。
- 内声音以 **楷体斜体**、**括弧**、**破折号插话**、或干脆融进叙述语气里。例子：

  > *（【内陆帝国】对你说——那只啤酒瓶上的水珠是它在哭。）*
  > 你把手放回来。

  或：

  > ——这只是一瓶，【食髓知味】低声道。
  > 你知道不是。

- **技能独白过长时**可以自成一段，用 **【技能名】** 领起。但一篇中这样做不超过三次。
- **思想内阁、流派**偶尔出场。一篇至多挂一到两条。
- 故事可以有**真正的其他角色**——店员、同事、老友、陌生人——他们的对白用正文写，不占用内声音的版面。
- **结尾要有一个安静的动作**，像《随笔循环》那样的收笔。不是爆点，是轻轻合上的门。

## 与《随笔循环》的关系

《随笔循环》是**一个声音试了三十种容器**。
《极乐独白》是**二十四种声音共用一只肉身**，去生活。

前者是文体实验。后者是**附身叙事**。
